Basic router configs 

Router#config terminal
Enter configuration commands, one per line.  End with CNTL/Z.
    !! configure a hostname
Router(config)#hostname dk-router
    !! set up an interface for management
dk-router(config)#interface Gigabit0/0
dk-router(config-if)#ip address 192.168.0.1 255.255.255.0
dk-router(config-if)#no shutdown
dk-router(config-if)#exit
    !! enable a console password
dk-router(config)#line console 0
dk-router(config-line)#password cisco
dk-router(config-line)#login
    !! enable telnet access
dk-router(config-line)#line vty 0 15
dk-router(config-line)#password cisco
dk-router(config-line)#login
dk-router(config-line)#exit
dk-router(config)#
